# sns_ai

Grok API と Claude Code を使った X（Twitter）投稿自動化ワークフロー。
AIリサーチ → 投稿下書き生成 → Typefully 予約投稿 までを一気通貫で行います。

---

## ディレクトリ構成

```
sns_ai/
├── 00_ナレッジ/          # ペルソナ・投稿スタイル・テンプレートなど固定知識
│   ├── マスター.md
│   ├── 価値観.md
│   ├── 執筆スタイル.md
│   ├── 最新コンテキスト.md
│   └── 投稿テンプレート/  # テンプレート型ごとの個別ファイル（01〜10）
├── 01_リサーチ/           # /research コマンドで生成された日次レポート（YYYY.MM.DD.md）
├── 02_アウトプット/       # 生成した投稿下書き
│   └── X投稿/
│       ├── 下書き/
│       └── 投稿済み/
├── 03_分析/               # パフォーマンス分析メモ（手動）
├── research/              # リサーチ処理の Python パッケージ
├── 参考/                  # 参考資料・マニュアル
├── .env                   # APIキー（Git管理外）
└── .env.example           # .env のサンプル
```

## セットアップ

### 1. 依存パッケージのインストール

```bash
pip install openai python-dotenv
```

### 2. 環境変数の設定

`.env.example` をコピーして `.env` を作成し、各APIキーを設定します。

```bash
cp .env.example .env
```

| 変数名 | 取得先 |
|---|---|
| `GROK_API_KEY` | https://console.x.ai |
| `TYPEFULLY_API_KEY` | https://typefully.com → Settings → API |

## 使い方

すべての操作は **Claude Code** 上のスラッシュコマンドで実行します。

### 投稿フロー

```
1. リサーチ実行（自動）
   /research

2. リサーチ結果を確認し、使うネタを選ぶ（手動）

3. 投稿を生成する（どちらか選択）
   /x-post <海外ツイートのURL>        # 海外ツイートを日本語スレッドに翻訳
   /buzz-post <トピック>               # トピック指定でバズ投稿を生成

4. Cursor上で内容を確認・修正（手動）

5. Typefully に下書きを送信
   /typefully-draft

6. Typefully で投稿予約（手動）
```

### コマンド一覧

| コマンド | 説明 |
|---|---|
| `/research` | Grok で X・HN・Reddit・Product Hunt・Google Trends を横断リサーチし、`01_リサーチ/` にレポートを保存 |
| `/x-post <URL>` | 指定した海外ツイートを日本語スレッド形式に翻訳・編集 |
| `/buzz-post <トピック>` | トピックからバズりやすいX投稿を生成 |
| `/typefully-draft` | 現在の下書きを Typefully に送信 |

## 投稿テンプレート

`00_ナレッジ/投稿テンプレート/` に10種類の型を用意しています。

| # | テンプレート | 目的 | 優先度 |
|---|---|---|---|
| 01 | 理想図を描かせる | 憧れ・ファン化 | ★★☆ |
| 02 | 再現性を担保する | 「自分にもできる」感 | ★★★ |
| 03 | 即行動を促す | 緊急行動 | ★☆☆ |
| 04 | 投資重要性を説く | 商品購買 | ★☆☆ |
| 05 | 新たな学びを提供する | フォロー促進 | ★★★ |
| 06 | 可能性を肯定する | モチベーション向上 | ★★☆ |
| 07 | 間接的に否定する | 差別化 | ★★☆ |
| 08 | 親近感を沸かせる | 距離を縮める | ★★★ |
| 09 | 向上心を表現する | 共に成長する仲間感 | ★★☆ |
| 10 | 権威性を表現する | 信頼・説得力 | ★★★ |

投稿生成時に型を明示的に指定することで、意図したトーンの投稿が得られます。

```
/buzz-post AIエージェント活用事例 --template 02_再現性を担保する
```
