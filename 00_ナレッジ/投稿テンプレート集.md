# 投稿テンプレート集.md

> 10の運用テンプレートを「信頼される会社アカウント」に翻訳した実践ガイドです。
> 各テンプレートの目的・使い所・具体的な投稿例をまとめています。

---

## 1. 理想図を描かせる投稿

**目的:** 「AIを使いこなせたらこうなれる」という具体的な未来を見せる

**AI驚き屋との違い:** 「AIで人生が変わる」ではなく「AI導入後に〇〇業務がどう変わったか」を具体的に見せる

**投稿例:**
```
AIを導入した会社の経理担当者から話を聞きました
月40時間かかっていた請求書処理が6時間になったとのこと
削った34時間を「新規顧客への提案書作成」に使い始めたそうです
時短ではなく「やれることが増えた」感覚が大きいと言っていました
```

---

## 2. 再現性を担保する投稿

**目的:** 「難しそうなAI活用が自分にもできる」と思わせる

**ポイント:** ハードルの低さ・具体的な手順・初期状態の正直な開示

**投稿例:**
```
半年前、私はChatGPTの使い方すら知りませんでした
最初にやったのはメールの返信文を書かせることだけ
そこから業務で試せることを毎週1つ増やした結果
今は週10時間分の作業を任せられています
始めるハードルは低い
```

---

## 3. 即行動を促す投稿

**目的:** 「今すぐ試してほしい」と思わせる（信頼構築後に解放）

**注意:** 煽りにならないよう、恐怖ではなく「今試すと得られるもの」を伝える

**投稿例:**
```
今日の夜30分だけ試してほしいことがあります
ChatGPTに「私の仕事は〇〇です。どんな業務を自動化できますか？」と聞いてみてください
5〜10個のアイデアが出てきます
そのうち1つだけ来週試す。それだけでOKです
```

---

## 4. 投資重要性を説く投稿

**目的:** AI学習・ツール導入への投資価値を伝える（信頼構築後に使う）

**注意:** 「高額塾に投資しろ」ではなく「時間・学習への投資」から入る

**投稿例:**
```
月2,000円のAIツールを1年間使い続けた場合
時間換算で月20時間削減できれば、時給2,000円でも年間48万円分
ツール代は年間24,000円
費用対効果の計算をしてから判断すると、導入の意思決定が早くなります
```

---

## 5. 新たな学びを提供する投稿

**目的:** 「このアカウントをフォローすると得する」と思わせる

**ポイント:** 普通の人が知らない視点・検証済みの具体的なノウハウ

**投稿例:**
```
AIへの指示で「箇条書きで教えて」より効果的な一言があります
「私が実際に使う前提で、手順を5ステップで書いてください」
すると操作可能な具体手順が返ってきます
「箇条書き」は形式の指定、「実際に使う前提で」は品質の指定
この違いが出力精度を変えます
```

---

## 6. 可能性を肯定する投稿

**目的:** AI活用に踏み出せていない読者を励ます

**ポイント:** 「あなたにもできる」を押しつけない。「こういう人が実際にできた」という事例で語る

**投稿例:**
```
「ITが苦手だからAIは無理」とよく聞きます
でも、私が見てきた中でAI活用が一番うまくいったのは
「毎日同じ作業を繰り返している」と気づいていた人でした
ITスキルより「どこに時間をとられているか」への気づきの方が大事です
```

---

## 7. 間接的に否定する投稿

**目的:** 一般的な誤解や非効率なやり方を否定し、正しいアプローチを示す

**ポイント:** 個人・競合を批判しない。「考え方」「方法論」に対して否定する

**投稿例:**
```
「AIツールをとりあえず全部試す」をやっていた時期があります
3ヶ月後に残ったのは2つだけ
今思うと、先に「自分の業務のどこが一番の時間泥棒か」を特定してから導入ツールを決めた方が早かった
ツール選びより課題の特定が先です
```

---

## 8. 親近感を沸かせる投稿

**目的:** 「完璧な専門家」ではなく「一緒に歩んでいる仲間」として認識してもらう

**ポイント:** 失敗・試行錯誤・等身大のエピソードを正直に語る

**投稿例:**
```
先週、AIに任せた資料が全然使えない状態で返ってきました
指示が曖昧だったのが原因です
「〇〇についてまとめて」ではなく「対象者・目的・形式・文字数」を指定して再挑戦
今度はほぼそのまま使えるものができました
失敗から学んだ指示の型をまとめます
```

---

## 9. 向上心を表現する投稿

**目的:** 「このアカウントと一緒に成長したい」と思わせる

**ポイント:** 目標・挑戦・悔しさなどのリアルな成長過程を見せる

**投稿例:**
```
今月から新しいことを始めました
AIエージェントを使った業務自動化の実験です
正直まだうまくいっていませんが、何が機能して何が失敗したかを記録しながら進めます
学んだことは随時シェアします
一緒に試してみたい方はフォローしていてください
```

---

## 10. 権威性を表現する投稿

**目的:** 「この人は信頼できる」という根拠を積み上げる

**ポイント:** 数字・実績・具体事例で語る。自己申告の「専門家です」は使わない

**投稿例:**
```
AI活用の相談を受けた会社で実施した改善を報告します
対象業務：週次レポート作成（従来4時間）
試した方法：Claude APIでテンプレート化＋データ自動入力
結果：45分に短縮
課題：イレギュラーな数字への対応はまだ手動
完璧ではありませんが、再現可能な手順として公開します
```

## 投稿の週間バランス例

| 曜日 | テンプレートタイプ |
|---|---|
| 月 | 新たな学びを提供する（実験・検証報告） |
| 火 | 親近感を沸かせる（失敗・等身大エピソード） |
| 水 | 再現性を担保する（手順・ステップ系） |
| 木 | 権威性を表現する（実績・数字報告） |
| 金 | 間接的に否定する or 理想図を描かせる |
| 土 | 向上心・可能性を肯定する |
| 日 | 投稿なし or まとめ・振り返り系 |
